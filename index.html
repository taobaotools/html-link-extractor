HTML Link Extractor

This website extracts all links given a URL or HTML - paste your HTML into the text area. Some websites don't allow you to get the HTML directly so you may have to do it yourself.
You can filter by 'http/https only' links and/or a keyword to search by. The website also supports converting mobile, world, h5 Taobao links to Chinese versions on the fly!

Expand (help)...

To copy the HTML on a web page, press CTRL + U (view page source) then CTRL + A (select all) then CTRL + C (copy) then CTRL + V (paste) in the HTML text area. Replace CTRL with the 'Command' key on a Mac.

The reason why some website URLs don't work here is that those websites block us from directly accessing the content HTML of their web pages using JavaScript AJAX Requests.
